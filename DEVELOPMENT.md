# 开发指南

## 环境要求

- Python 3.8+
- pip 或 conda

## 安装步骤

1. 克隆仓库
`ash
git clone https://github.com/Din829/KD0825.git
cd KD0825
`

2. 安装依赖
`ash
pip install -r requirements.txt
`

3. 配置环境变量
`ash
cp .env.example .env
# 编辑 .env 文件，添加你的API密钥
`

## 开发流程

1. 创建新分支
2. 进行开发
3. 提交更改
4. 创建Pull Request
5. 代码审查
6. 合并到主分支

## 代码规范

- 使用Python PEP 8代码风格
- 添加适当的注释和文档字符串
- 编写单元测试
